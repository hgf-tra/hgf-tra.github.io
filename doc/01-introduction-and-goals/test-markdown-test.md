## Introduction

Markdown is a lightweight markup language that allows you to format text and prepare it for the web. It's easy to learn and super useful!

## Bold and Italics

You can make text **bold** or *italic* by enclosing it accordingly.

- **Bold:** `**Text**`
- *Italic:* `*Text*`

## Lists
